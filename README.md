# AI-SDLC Mob Programming Testing

A repository dedicated to AI-powered Software Development Lifecycle (AI-SDLC) practices, frameworks, playbooks, and reference architectures.

This repository captures reusable assets for:

- AI-assisted requirements engineering
- Solution architecture and design
- Mob programming facilitation
- Implementation planning
- Testing and quality engineering
- DevOps and delivery automation
- Governance and review practices
- Domain-specific SDLC playbooks

## Repository Structure

```text
AI-SDLC-Mob-Programming-Testing/
├── skills/
├── requirements-engineering/
├── solution-architecture/
├── mob-programming/
├── implementation/
├── testing/
├── devops/
├── governance/
├── domains/
│   └── shopify-headless/
└── prompts/
```

## Starting Domain

The first domain implementation is **Shopify Headless Middleware**, focused on mob programming, middleware/API architecture, commerce workflows, and testing strategies.

## Purpose

The goal is to evolve this into a reusable AI-SDLC knowledge base that helps teams move from business use case to collaborative design, implementation planning, testing, and delivery.
