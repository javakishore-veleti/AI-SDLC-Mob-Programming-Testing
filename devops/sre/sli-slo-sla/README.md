# SLI, SLO and SLA Design

## Definitions

### SLI
Measurement of system behavior.

Examples:

- Availability
- Latency
- Error rate

### SLO
Target objective.

Examples:

- 99.9% uptime
- P95 latency < 300ms

### SLA
Business commitment.

## Mob Programming Outputs

- Service catalog
- Reliability targets
- Error budgets
- Executive reporting model
